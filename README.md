# Resume Review

**Resume Review** is an intelligent resume enhancement tool that leverages the power of **Large Language Models (LLMs)** to analyze and refine resumes.  
This **Streamlit-based application** allows users to upload their resumes in PDF format, optionally include a job description, and receive clear, section-by-section feedback with actionable suggestions for improvement.

---

## Features
- **Smart Resume Parsing:** Automatically extracts and structures text from PDF resumes.  
- **AI-Driven Review:** Analyzes resume content with or without a provided job description.  
- **Section-Level Insights:** Generates detailed feedback for each part of the resume.  
- **Improvement Guidance:** Suggests revisions with labeled impact levels.  
- **Interactive UI:** Simple Streamlit interface with side-by-side comparison and navigation controls.
